# Get-PaConfigDiff

## Synopsis

Checks to see if there's a difference in the candidate and running configuration on a Palo Alto Device.

## Syntax


```powershell
Get-PaConfigDiff 
```

## Description

Checks to see if there's a difference in the candidate and running configuration on a Palo Alto Device.
